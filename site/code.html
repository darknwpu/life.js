• 编码

    统一用 utf-8

  • 长度

    长度不超过 80 个字符。别小看这一条规则，如果严格去遵循，你会发现代码变清晰了。而且，你一定能做到的。

  • 缩进

    缩进使用 4个 空格。

  • 花括号

    缩进使用 4个 空格。

  • 缩进

    花括号不换行

    好

    if (foo) {

}

    坏

    if (foo)
{

}

    不允许一行判断，一律换行

    坏

    if (foo) return;

  • 空格

    操作符之间需要空格

    好

    var x = y + z

    坏

    var x=y+z

    只空一格

    好

    {
     a: 'short',
     looooongname: 'long'
}

    坏

    {
    a           : 'short',
    looooongname: 'long'
}

  • 逗号与换行

    自然人的处理方法，不要以操作符开头，将操作符放在一行最后。

    好

    {
    a: 'a',
    b: 'b',
    c: 'c'
}

    var str = "aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa" +
            "bbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbb" +
            "ccccccccccccccccccccccccccccccccccccccccccccccccc";

    坏

    {
    a: 'a'
    ,b: 'b'
    ,c: 'c'
}

    var str = "aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"
            + "bbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbb"
            + "ccccccccccccccccccccccccccccccccccccccccccccccc";

  • 多变量写法

    使用 var 风格，一个变量一个var

    好

    var $ = require('jquery');
var position = require('position');
var useIframe = $.browser.msie && Number($.browser.version) < 7;

    坏

    var cache, event, all, list, i, len, rest = [], args;

  • 文档中，中英文夹杂时的风格

    含有中文时，用中文全角标点

    中文和英文之间，空一个英文空格
